## Design Context

### Users
Mixed audience of technical and non-technical users interacting with an AI agent chat interface. Users range from developers automating browser tasks to business professionals creating workflows. They need to feel confident the agent is working correctly while maintaining a sense of control. The interface should balance power with accessibility — never intimidating, never dumbed-down.

### Brand Personality
**Warm & Approachable** — friendly, inviting, human. Three words: *helpful, clear, trustworthy*.

The interface should feel like a capable assistant that communicates its state transparently. Warmth comes from thoughtful micro-interactions and clear feedback, not from decoration.

### Aesthetic Direction
- **Visual tone:** Clean and warm, inspired by Notion/Slack's approachability with Linear/Vercel's polish
- **References:** ChatGPT/Claude.ai (conversational AI patterns), Linear/Vercel (refined typography and motion), Slack/Notion (structured warmth)
- **Anti-references:** Avoid anything that feels cold/clinical or overly futuristic
- **Theme:** Support both light and dark modes, host app controls which is active
- **Colors:** Muted, professional palette. Avoid neon/saturated hues. Avoid gradients. Current palette uses Slate grays with status colors (green=ready, blue=working, red=error, amber=waiting). Brand navy `#292656` appears in iconography.
- **Typography:** System/default font stacks, rem-based sizing, medium weights
- **Icons:** Lucide React for standard icons, custom SVGs for brand-specific elements
- **Motion:** Framer Motion for meaningful transitions — purposeful, not decorative

### Design Principles
1. **Clarity over cleverness** — Every element should communicate its purpose immediately. Status, state, and actions must be unambiguous.
2. **Warm minimalism** — Strip away the unnecessary, but keep enough warmth (rounded corners, soft shadows, gentle transitions) that the interface feels human.
3. **Transparent agency** — The agent's state should always be visible and honest. Users should never wonder what the agent is doing.
4. **Accessible by default** — WCAG AA compliance. Good contrast ratios, keyboard navigation, screen reader support, and reduced-motion considerations.
5. **Adaptive, not duplicated** — Single design system that adapts to light/dark themes through CSS variables or Tailwind's dark mode, not separate implementations.
